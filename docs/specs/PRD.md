# LangMap 產品需求文檔 (PRD)

## 1. 產品概述
LangMap 是一個展示語言豐富度差異的多端（Web & iOS）平臺，提供多來源語料（權威、AI、用戶）的查詢、瀏覽與可視化功能。

## 2. 核心概念與數據來源
### 2.1 語料來源等級
1. **權威數據 (Authoritative)**：來自書籍、影視、音樂作品。
2. **AI 補充 (AI Generated)**：由 AI 生成，帶標識，支持點讚/點踩。
3. **用戶提交 (User Contributed)**：用戶貢獻，經 AI 審核，支持點讚/點踩。

### 2.2 地域細分
精細到：`[語言]-[地區][鄉鎮]`（如：閩南語-汕頭澄海）。
- 最小粒度爲鄉鎮。

## 3. 功能需求

### 3.1 首頁與熱力圖 (Home & Heatmap)
- **可視化**：地圖展示各語言/地區語料豐富度。
- **統計**：實時顯示表達式總數、涉及語言數及地域覆蓋數。

### 3.2 詞條查詢 (Search)
- **多維度檢索**：支持關鍵詞、來源語言、地域過濾。
- **AI 補全**：無命中時可觸發 AI 實時生成。
- **外部跳轉**：提供 TTS 或 Forvo 等外部朗讀鏈接。

### 3.3 語言地圖 (Language Map)
- **對比視圖**：在地圖上展示同一詞彙在不同地理位置的表達差異。

### 3.4 集合與收藏 (Collections)
- **個人收藏**：用戶可創建公開或私有的表達式集合。

### 3.5 UI 翻譯系統 (UI Translation)
- **動態加載**：支持新語言上線，達到一定翻譯進度（如 60%）後自動激活。
- **社區貢獻**：允許用戶在線翻譯系統的 UI 文本。

### 3.6 異步導出系統 (Export)
- **長時任務**：支持將大量語料異步導出爲 ZIP/Excel 等格式。

### 3.7 批量語義提交 (Batch Submission)
- **跨語言提交**：支持一次性提交一組共享同一語義的多語言表達式。
- **自動關聯**：系統自動處理重複內容並建立與語義組的關聯。
- **更新傳播**：詞句內容更正或歸屬調整時，自動維持或更新關聯網絡。

### 3.8 移動端 (iOS App)
- **便攜查閱**：適配手機端的地圖與搜索體驗。
- **輔助錄入**：支持移動端快速提交新語料。

## 4. 界面需求 (UI Requirements)
- **簡潔直觀**：地圖操作流暢，搜索響應快。
- **明確標識**：來源標籤（權威/AI/用戶）在 UI 上必須顯眼。
- **國際化**：支持動態切換界面語言。

## 5. 非功能需求
- **性能**：地圖加載速度優化。
- **可靠性**：AI 審核與人工審核相結合保證語料質量。
