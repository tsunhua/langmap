# LangMap 产品需求文档 (PRD)

## 1. 产品概述
LangMap 是一个展示语言丰富度差异的多端（Web & iOS）平台，提供多来源语料（权威、AI、用户）的查询、浏览与可视化功能。

## 2. 核心概念与数据来源
### 2.1 语料来源等级
1. **权威数据 (Authoritative)**：来自书籍、影视、音乐作品。
2. **AI 补充 (AI Generated)**：由 AI 生成，带标识，支持点赞/点踩。
3. **用户提交 (User Contributed)**：用户贡献，经 AI 审核，支持点赞/点踩。

### 2.2 地域细分
精细到：`[语言]-[地区][乡镇]`（如：闽南语-汕头澄海）。
- 最小粒度为乡镇。

## 3. 功能需求

### 3.1 首页与热力图 (Home & Heatmap)
- **可视化**：地图展示各语言/地区语料丰富度。
- **统计**：实时显示表达式总数、涉及语言数及地域覆盖数。

### 3.2 词条查询 (Search)
- **多维度检索**：支持关键词、来源语言、地域过滤。
- **AI 补全**：无命中时可触发 AI 实时生成。
- **外部跳转**：提供 TTS 或 Forvo 等外部朗读链接。

### 3.3 语言地图 (Language Map)
- **对比视图**：在地图上展示同一词汇在不同地理位置的表达差异。

### 3.4 集合与收藏 (Collections)
- **个人收藏**：用户可创建公开或私有的表达式集合。

### 3.5 UI 翻译系统 (UI Translation)
- **动态加载**：支持新语言上线，达到一定翻译进度（如 60%）后自动激活。
- **社区贡献**：允许用户在线翻译系统的 UI 文本。

### 3.6 异步导出系统 (Export)
- **长时任务**：支持将大量语料异步导出为 ZIP/Excel 等格式。

### 3.7 批量语义提交 (Batch Submission)
- **跨语言提交**：支持一次性提交一组共享同一语义的多语言表达式。
- **自动关联**：系统自动处理重复内容并建立与语义组的关联。
- **更新传播**：词句内容更正或归属调整时，自动维持或更新关联网络。

### 3.8 移动端 (iOS App)
- **便携查阅**：适配手机端的地图与搜索体验。
- **辅助录入**：支持移动端快速提交新语料。

## 4. 界面需求 (UI Requirements)
- **简洁直观**：地图操作流畅，搜索响应快。
- **明确标识**：来源标签（权威/AI/用户）在 UI 上必须显眼。
- **国际化**：支持动态切换界面语言。

## 5. 非功能需求
- **性能**：地图加载速度优化。
- **可靠性**：AI 审核与人工审核相结合保证语料质量。
