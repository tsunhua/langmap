# 关于 LangMap

## 项目目的

LangMap 是一个开源、社区驱动的在线语言地图项目，致力于收集世界各地的语言短语和表达方式，展示不同语言之间的差异，并为语言爱好者提供有价值的资源。我们的目标是建立一个全球语言表达的共享平台，让每个人都能轻松学习和探索各种语言的独特之处。

通过 LangMap，我们希望：
- 保护和传承濒危语言和方言
- 展示同一概念在不同语言中的多样表达
- 促进跨文化交流和理解
- 为语言研究者和爱好者提供实用工具

## 基本使用操作

### 浏览表达内容
1. 访问主页浏览最新添加的语言表达
2. 使用搜索功能查找特定词语或短语
3. 点击表达卡片查看详细信息和历史版本

### 添加新表达（需要注册账户）
1. 点击右上角的"登录"按钮进行注册或登录
2. 登录后，在主页点击"添加新表达"按钮
3. 填写表达的基本信息，包括：
   - 选择语言
   - 输入词语或短语
   - 提供定义和用法说明
   - 添加例句
4. 提交审核，等待管理员确认

### 切换语言界面
1. 在页面右上角找到语言切换下拉菜单
2. 选择您偏好的界面语言
3. 系统会自动保存您的语言偏好设置
