# LangMap 业务需求文档 (BRD)

## 1. 项目愿景 (Project Vision)
LangMap 是一个开源、社区驱动的多端（Web & iOS）语言地图项目，致力于收集世界各地的语言短语和表达方式，展示不同语言之间的差异。我们的目标是建立一个全球语言表达的共享平台，让每个人都能轻松学习和探索各种语言的独特之处。

## 2. 市场背景与机会 (Market Context & Opportunities)
目前的语言类产品要么学术性强但不够可交互（如 WALS），要么是社区驱动但缺少严格来源标注（如 Tatoeba）。LangMap 的机会在于：
- **地域细粒度**：支持细化到乡镇级别的语言分布。
- **跨平台体验**：提供 Web 端的大屏分析与 iOS 端的便携查阅/贡献。
- **来源透明化**：显示权威、AI 补充与用户提交的明显差异。

## 3. 目标用户画像 (Target Audience)
- **语言学研究者与田野工作者**：利用 Web 端进行深度数据管理，利用 iOS 端在田野调查中快速记录。
- **语言学习者与文化爱好者**：希望通过手机端随时学习地道发音。
- **翻译 / 本地化工程师**：需要多语对比与地域变体信息。
- **文化机构**：保存濒危语言样本，展示语言多样性。

## 4. 业务目标 (Business Objectives)
- **保护核心**：保护和传承濒危语言和方言。
- **促进交流**：展示同一概念在不同语言中的多样表达，促进跨文化理解。
- **提供工具**：为语言研究者和爱好者提供实用、直观的工具。

## 5. 商业模式建议 (Optional Business Models)
- **数据授权 / API 付费**：向企业、研究机构提供结构化数据访问。
- **捐赠与研究资助**：通过非营利机构或学术项目获取资金。
- **SaaS 服务**：提供高级可视化、批量导出、私有数据集支持。
