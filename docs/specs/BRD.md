# LangMap 業務需求文檔 (BRD)

## 1. 項目願景 (Project Vision)
LangMap 是一個開源、社區驅動的多端（Web & iOS）語言地圖項目，致力於收集世界各地的語言短語和表達方式，展示不同語言之間的差異。我們的目標是建立一個全球語言表達的共享平臺，讓每個人都能輕鬆學習和探索各種語言的獨特之處。

## 2. 市場背景與機會 (Market Context & Opportunities)
目前的語言類產品要麼學術性強但不夠可交互（如 WALS），要麼是社區驅動但缺少嚴格來源標註（如 Tatoeba）。LangMap 的機會在於：
- **地域細粒度**：支持細化到鄉鎮級別的語言分布。
- **跨平臺體驗**：提供 Web 端的大屏分析與 iOS 端的便攜查閱/貢獻。
- **來源透明化**：顯示權威、AI 補充與用戶提交的明顯差異。

## 3. 目標用戶畫像 (Target Audience)
- **語言學研究者與田野工作者**：利用 Web 端進行深度數據管理，利用 iOS 端在田野調查中快速記錄。
- **語言學習者與文化愛好者**：希望通過手機端隨時學習地道發音。
- **翻譯 / 本地化工程師**：需要多語對比與地域變體信息。
- **文化機構**：保存瀕危語言樣本，展示語言多樣性。

## 4. 業務目標 (Business Objectives)
- **保護核心**：保護和傳承瀕危語言和方言。
- **促進交流**：展示同一概念在不同語言中的多樣表達，促進跨文化理解。
- **提供工具**：爲語言研究者和愛好者提供實用、直觀的工具。

## 5. 商業模式建議 (Optional Business Models)
- **數據授權 / API 付費**：向企業、研究機構提供結構化數據訪問。
- **捐贈與研究資助**：通過非營利機構或學術項目獲取資金。
- **SaaS 服務**：提供高級可視化、批量導出、私有數據集支持。
