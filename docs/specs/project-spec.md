# 粗略的需求
构建一个语言地图网站，显示出各语言丰富度的差异。
语料的来源有三种：
1. 权威书籍、影视、音乐作品。
2. AI 补充（要有AI标识，并提供赞和踩按钮）。
3. 用户提交（需要AI审核，并提供赞和踩按钮）。

该网站包含以下页面：
1. 首页：创造性地显示各语言的丰富度差异。
2. 查询词条页面，支持以A语言查询到任意语言的表达（当在语料库中无结果时，支持让用户点击AI补充或提交补全），并支持点击跳转到外部软件进行朗读。如果命中语料库，则可选择进入语言地图页面。
3. 语言地图，支持查询某一词/句在各语言的表达。

语言要可细化到地区（`[语言]-[国家城市][乡镇]`），充分显示出语言的差异，例如：闽南语-中国汕头澄海，国家地区应通过地理位置选择器选择（默认用户当前位置）而不是用户输入，并最多只截取到乡镇。

# 粗略的架构

前端：采用 Vue3
后端：采用 Python3.12
