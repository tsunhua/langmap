# 政策与合规文档

本目录包含 LangMap 平台的法律政策文档。

## 文档列表

- [terms-privacy-content.md](./terms-privacy-content.md) - 服务条款、隐私政策与内容政策（英文版）
- [terms-privacy-content-zh.md](./terms-privacy-content-zh.md) - 服务条款、隐私政策与内容政策（中文版）

## 文档说明

### 服务条款 (Terms of Service)
定义用户使用 LangMap 平台的法律条款，包括：
- 使用资格
- 用户账户责任
- 用户生成内容的授权许可
- 禁止行为
- 内容移除机制
- 平台知识产权
- 服务免责声明

### 隐私政策 (Privacy Policy)
说明 LangMap 如何收集、使用和保护用户信息，包括：
- 收集的信息类型
- 信息使用方式
- Cookie 政策
- 信息分享对象
- 数据保留策略
- 数据安全措施
- 用户权利（GDPR/CCPA）
- 儿童隐私保护

### 内容政策 (Content Policy)
定义 LangMap 上允许和禁止的内容类型，包括：
- 可接受的内容
- 禁止内容（非法、不当行为、低质量）
- 版权政策
- 内容审核流程
- 申诉机制

## 更新

政策文档的生效日期在文档开头标注。任何政策更新会在网站公告，继续使用平台即代表接受更新内容。

## 联系方式

如有政策相关疑问，请通过平台提供的联系方式与我们联系。
